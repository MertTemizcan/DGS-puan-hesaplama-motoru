# DGS Puan Hesaplama Motoru / DGS Score Calculator

English Description

This is a Java-based console application developed for DGS (Vertical Transfer Exam)
candidates to calculate their estimated numerical scores. The software calculates net scores
based on user-provided correct and incorrect answers and provides an estimated placement
score using current coefficients

Türkçe Açıklama

Dikey Geçiş Sınavı (DGS) adayları için geliştirilmiş, sayısal ağırlıklı puan hesaplaması yapan bir
Java konsol uygulamasıdır. Bu yazılım, kullanıcıdan alınan doğru ve yanlış sayılarını kullanarak
netleri hesaplar ve güncel katsayılar üzerinden tahmini bir yerleştirme puanı sunar.
